<h2 style="text-align: center;"> Γενικά </h2>

Γνωρίζουμε πολλούς χειριστές από το σχολείο. Είναι πράγματα όπως η πρόσθεση +, ο πολλαπλασιασμός *, η αφαίρεση - και ούτω καθεξής.  
Σε αυτό το κεφάλαιο, θα ξεκινήσουμε με απλούς τελεστές και στη συνέχεια θα επικεντρωθούμε σε πτυχές που αφορούν τη JavaScript και δεν καλύπτονται από την αριθμητική του σχολείου.  

---

<h2 style="text-align: center;"> Μαθηματικά </h2>
Υποστηρίζονται οι ακόλουθες μαθηματικές πράξεις:

| Σύμβολο | Πράξη                       |
|---------|-----------------------------|
| +       | Πρόσθεση                    |
| -       | Αφαίρεση                    |
| *       | Πολλαπλασιασμό              |
| /       | Διαίρεση                    |
| **      | Ύψωση σε δύναμη             |
| %       | Υπόλοιπο ακέραιας διαίρεσης |

```js
alert(5 + 2) // 7
alert(5 - 2) // 3
alert(5 * 2) // 10
alert(10 / 2) // 5
```

<h3 style="text-align: center;">Υπόλοιπο</h3>
Ο τελεστής υπολοίπου %, παρά την εμφάνισή του, δεν σχετίζεται με τα ποσοστά.  
Το αποτέλεσμα του a % b είναι το υπόλοιπο της ακέραιης διαίρεσης του a με το b.

```js
alert( 5 % 2 ); // 1
alert( 8 % 3 ); // 2
alert( 8 % 4 ); // 0
```

<h3 style="text-align: center;">Δυνάμεις</h3>
Ο τελεστής πολλαπλασιασμού ```a ** b``` αυξάνει το a στη δύναμη του b.  
Στα μαθηματικά του σχολείου, το γράφουμε ως <Latex formula="a^b"/>.

```js
alert( 2 ** 2 ); // 2² = 4
alert( 2 ** 3 ); // 2³ = 8
alert( 2 ** 4 ); // 2⁴ = 16
```

Ακριβώς όπως στα μαθηματικά, ο τελεστής εκθετικοποίησης ορίζεται και για μη ακέραιους αριθμούς.  
Για παράδειγμα, η τετραγωνική ρίζα είναι ένας πολλαπλασιασμός με <Latex formula="\frac{1}{2}" />:

```js
alert( 4 ** (1/2) ); // 2 (η δύναμη του 1/2 είναι η ίδια με την τετραγωνική ρίζα)
alert( 8 ** (1/3) ); // 2 (η δύναμη του 1/3 είναι η ίδια με την κυβική ρίζα)
```

<h2 style="text-align: center;">Σύνδεση αλφαριθμητικών με ```+```</h2>
Ας γνωρίσουμε τα χαρακτηριστικά των τελεστών της JavaScript που ξεπερνούν την αριθμητική του σχολείου.  
Συνήθως, ο τελεστής συν + αθροίζει αριθμούς.  
Όμως, αν ο τελεστής ```+``` εφαρμοστεί σε συμβολοσειρές, τις συγχωνεύει:  

```js
let s = "my" + "string";
alert(s); // mystring
```

Σημειώστε ότι αν κάποιο από τα τελεστέα είναι συμβολοσειρά, τότε και το άλλο μετατρέπεται σε συμβολοσειρά.

```js
alert( '1' + 2 ); // "12"
alert( 2 + '1' ); // "21"
```

Βλέπετε, δεν έχει σημασία αν ο πρώτος τελεστής είναι συμβολοσειρά ή ο δεύτερος.  
Ακολουθεί επίσης ένα πιο σύνθετο παράδειγμα.

```js
alert(2 + 2 + '1' ); // "41" όχι "221"
```

<h2 style="text-align: center;">Τροποποίηση επί τόπου</h2>
Συχνά χρειάζεται να εφαρμόσουμε έναν τελεστή σε μια μεταβλητή και να αποθηκεύσουμε το νέο αποτέλεσμα στην ίδια μεταβλητή.  
Για παράδειγμα:

```js
let n = 2;
n = n + 5;
n = n * 2;
n = n - 1;
n = n / 2;
```

Αυτός ο συμβολισμός μπορεί να συντομευτεί χρησιμοποιώντας τους τελεστές ```+=```, ```-=```, ```*=```, ```/=```:

```js
let n = 2;
n += 5;
n *= 2;
n -= 1;
n /= 2;
```

<h2 style="text-align: center;">Αύξηση/μείωση</h2>

Η αύξηση ή η μείωση ενός αριθμού κατά ένα είναι από τις πιο συνηθισμένες αριθμητικές πράξεις.  
Έτσι, υπάρχουν ειδικοί τελεστές γι' αυτό:
+ Αύξηση, το ```++``` αυξάνει μια μεταβλητή κατά 1
+ Μείωση, το ```--``` μειώνει μια μεταβλητή κατά 1

```js
let a = 5;

alert(a); // 5
a++;
alert(a); // 6
a--;
alert(a) // 5
```
