<h2 style="text-align: center;"> Η ```<script> </script>``` tag</h2>

Τα προγράμματα JavaScript μπορούν να εισαχθούν σχεδόν οπουδήποτε σε ένα έγγραφο HTML χρησιμοποιώντας την ετικέτα ```<script>```.
Για αυτό το μάθημα θα χρησιμοποιήσουμε τον ενσωματωμένο editor που βλέπετε στο κέντρο του παραθύρου.
Αυτο το editor λειτουργεί σαν μια ετικέτα* ```<script>```.
Για να "τρέξεις" το editor πάτα το κουμπί, Τρέξε!, που βρίσκεται στα αριστερά.

<h3 style="text-align: center;"> Παραδειγμα: </h3>
```html
<!DOCTYPE HTML>
<html>

<body>

  <p>Πρίν...</p>

  <script>
    alert('Hello, world!');
  </script>

  <p>...Μετά</p>

</body>

</html>
```

***ετικέτα = tag**

---

<h2 style="text-align: center;"> ```alert()``` </h2>

Το ```alert('msg')``` δίνει εντολή στο πρόγραμμα περιήγησης να εμφανίσει ένα παράθυρο διαλόγου με ένα προαιρετικό μήνυμα, για αυτή την περίπτωση 'msg', και να περιμένει μέχρι ο χρήστης να κλείσει το παράθυρο διαλόγου.  
[Περισσότερες Πληροφορίες (ΑΓΓΛΙΚΑ)](https://developer.mozilla.org/en-US/docs/Web/API/Window/alert)
  
<h3 style="text-align: center;"> Παραδειγμα: </h3>
```js
alert("Hello!")
```

---

<h2 style="text-align: center;">console.log()</h2>
Η μέθοδος ```console.log()``` εξάγει ένα μήνυμα στην κονσόλα ιστού ή στην κονσόλα των επεξεργαστών στα δεξιά. Το μήνυμα μπορεί να είναι ένα απλό αλφαριθμητικό (με προαιρετικές τιμές αντικατάστασης) ή μπορεί να είναι ένα ή περισσότερα αντικείμενα JavaScript.  
[Περισσότερες Πληροφορίες (ΑΓΓΛΙΚΑ)](https://developer.mozilla.org/en-US/docs/Web/API/console/log)
  
Παράδειγμα
```console.log("Hello!")```
