<h2 style="text-align: center;"> Γενικά </h2>

Γνωρίζουμε πολλούς τελεστές σύγκρισης από τα μαθηματικά.

Στη JavaScript γράφονται ως εξής:

- Μεγαλύτερο/μικρότερο από: `a > b`, `a < b`.
- Μεγαλύτερο/μικρότερο ή ίσο: `α >= b`, `α <= b`.
- Ισοδυναμεί: Σημειώστε ότι το διπλό σύμβολο ισότητας == σημαίνει τον έλεγχο ισότητας, ενώ το απλό σύμβολο `a = b` σημαίνει ανάθεση.
- Δεν ισούται: Στα μαθηματικά ο συμβολισμός είναι <Latex formula="\neq" />, αλλά στη JavaScript γράφεται ως `a != b`.

Σε αυτό το μάθημα θα μάθουμε περισσότερα για τους διάφορους τύπους συγκρίσεων, τον τρόπο με τον οποίο η JavaScript τις πραγματοποιεί, συμπεριλαμβανομένων σημαντικών ιδιαιτεροτήτων.

---

<h2 style="text-align: center;"> Boolean είναι το αποτέλεσμα </h2>

Όλοι οι τελεστές σύγκρισης επιστρέφουν μια boolean τιμή:

- true - σημαίνει "ναι", "σωστό" ή "η αλήθεια".
- false - σημαίνει "όχι", "λάθος" ή "όχι η αλήθεια".

```js
alert(2 > 1); // true (correct)
alert(2 == 1); // false (wrong)
alert(2 != 1); // true (correct)
```

---

<h2 style="text-align: center;"> Αυστηρή ισότητα </h2>

Ένας κανονικός έλεγχος ισότητας == έχει ένα πρόβλημα. Δεν μπορεί να διακρίνει το 0 από το false:

```js
alert(0 == false); // true
alert("" == false); // true
```

Αυτό συμβαίνει επειδή οι τελεστές διαφορετικών τύπων μετατρέπονται σε αριθμούς με τον τελεστή ισότητας ==. Ένα κενό αλφαριθμητικό, όπως και το false, γίνεται μηδέν.

Τι πρέπει να κάνουμε αν θέλουμε να διαφοροποιήσουμε το 0 από το false;

** Ένας τελεστής αυστηρής ισότητας === ελέγχει την ισότητα χωρίς μετατροπή τύπου. **

Με άλλα λόγια, εάν τα a και b είναι διαφορετικών τύπων, τότε το a === b επιστρέφει αμέσως false χωρίς προσπάθεια μετατροπής τους.

---

<h2 style="text-align: center;"> Αποφυγή προβλημάτων </h2>

- Αντιμετωπίστε κάθε σύγκριση με `undefined`/`null` εκτός από την αυστηρή ισότητα `===` με εξαιρετική προσοχή.
- Μην χρησιμοποιείτε συγκρίσεις, `>=`, `>`, `<`, `<=`, με μια μεταβλητή που μπορεί να είναι `null`/`undefined`, εκτός αν είστε πραγματικά σίγουροι για το τι κάνετε. Αν μια μεταβλητή **μπορεί** να έχει αυτές τις τιμές, ελέγξτε τις ξεχωριστά.
