<h2 style="text-align: center;">Γενικά</h2>

Τις περισσότερες φορές, οι τελεστές και οι συναρτήσεις μετατρέπουν αυτόματα τις τιμές που τους δίνονται στο σωστό τύπο.  
Για παράδειγμα, η alert μετατρέπει αυτόματα οποιαδήποτε τιμή σε συμβολοσειρά για να την εμφανίσει. Οι μαθηματικές πράξεις μετατρέπουν τις τιμές σε αριθμούς.  
Υπάρχουν επίσης περιπτώσεις που χρειάζεται να μετατρέψουμε ρητά μια τιμή στον αναμενόμενο τύπο.  

---

<h2 style="text-align: center;">Μετατροπή String (συμβολοσειρών)</h2>

Η μετατροπή συμβολοσειράς συμβαίνει όταν χρειαζόμαστε τη μορφή συμβολοσειράς μιας τιμής.  
Για παράδειγμα, το `alert(value)` το κάνει για να εμφανίσει την τιμή.  
Μπορούμε επίσης να καλέσουμε τη συνάρτηση `String(value)` για να μετατρέψουμε μια τιμή σε συμβολοσειρά:  

Η μετατροπή συμβολοσειρών είναι ως επί το πλείστον προφανής. Το false γίνεται "false", το null γίνεται "null", κ.τλπ.  

```js
let value = true;
alert(typeof value); // boolean

value = String(value); // "true"
alert(typeof value); // string
```

---

<h2 style="text-align: center;">Μετατροπή αριθμών</h2>

Η αριθμητική μετατροπή σε μαθηματικές συναρτήσεις και εκφράσεις γίνεται αυτόματα.  
Για παράδειγμα, όταν η διαίρεση / εφαρμόζεται σε μη αριθμούς:  

```js
alert( "6" / "2" ); // 3
```

Μπορούμε να χρησιμοποιήσουμε τη συνάρτηση Number(value) για να μετατρέψουμε ρητά μια τιμή σε αριθμό:  

```js
let str = "123"; // "123"
alert(typeof str); // string

let num = Number(str); // 123

alert(typeof num); // number
```

Η ρητή μετατροπή απαιτείται συνήθως όταν διαβάζουμε μια τιμή από μια πηγή που βασίζεται σε συμβολοσειρά, όπως μια φόρμα κειμένου, αλλά περιμένουμε να εισαχθεί ένας αριθμός.  
Εάν η συμβολοσειρά δεν είναι έγκυρος αριθμός, το αποτέλεσμα μιας τέτοιας μετατροπής είναι NaN. Για παράδειγμα:  

```js
let age = Number("hello");
alert(age); // NaN

alert( Number("   123   ") ); // 123
alert( Number("123z") );      // NaN
alert( Number(true) );        // 1
alert( Number(false) );       // 0
```

<h2 style="text-align: center;">Μετατροπή Boolean</h2>

Η μετατροπή Boolean είναι η απλούστερη.  
Συμβαίνει στις λογικές πράξεις (αργότερα θα συναντήσουμε δοκιμές συνθηκών και άλλα παρόμοια πράγματα) αλλά μπορεί επίσης να εκτελεστεί ρητά με μια κλήση της Boolean(value).  

```js
alert( Boolean(1) ); // true
alert( Boolean(0) ); // false

alert( Boolean("hello") ); // true
alert( Boolean("") ); // false
```
