<h2 style="text-align: center;"> Γενικά </h2>

Υπάρχουν τέσσερις λογικοί τελεστές στη JavaScript: `||` (Ή), ` &&` (ΚΑΙ), `!` (ΌΧΙ).

Αν και ονομάζονται "λογικοί", μπορούν να εφαρμοστούν σε τιμές οποιουδήποτε τύπου, όχι μόνο boolean. Το αποτέλεσμά τους μπορεί επίσης να είναι οποιουδήποτε τύπου.

Ας δούμε τις λεπτομέρειες.

---

<h2 style="text-align: center;"> || </h2>

Ο τελεστής "OR" αναπαρίσταται με δύο σύμβολα κάθετης γραμμής:

```js
let result = a || b;
```

Στον κλασικό προγραμματισμό, το λογικό `Ή` προορίζεται για τον χειρισμό μόνο των boolean τιμών. Εάν κάποιο από τα ορίσματά του είναι αληθές, επιστρέφει αληθές, διαφορετικά επιστρέφει ψευδές.

Στη JavaScript, ο τελεστής είναι λίγο πιο δύσκολος και πιο ισχυρός. Αλλά πρώτα, ας δούμε τι συμβαίνει με τις boolean τιμές.

Υπάρχουν τέσσερις πιθανοί λογικοί συνδυασμοί:

```js
alert(true || true);    // true
alert(false || true);   // true
alert(true || false);   // true
alert(false || false);  // false
```

Όπως βλέπουμε, το αποτέλεσμα είναι πάντα αληθές, εκτός από την περίπτωση που και οι δύο τελεστές είναι ψευδείς.

Εάν ένας τελεστής δεν είναι boolean, μετατρέπεται σε boolean για την αξιολόγηση.

Για παράδειγμα, ο αριθμός 1 αντιμετωπίζεται ως αληθής, ενώ ο αριθμός 0 ως ψευδής:

```js
if (1 || 0) {
  // ( true || false )
  alert("truthy!");
}
```

Τις περισσότερες φορές, η εντολή OR `||` χρησιμοποιείται σε μια εντολή `if` για να ελέγξει αν κάποια από τις δεδομένες συνθήκες είναι αληθής.

```js
let hour = 9;

if (hour < 10 || hour > 18) {
  alert("Το γραφείο είναι κλειστό.");
}
```

```js
let hour = 12;
let isWeekend = true;

if (hour < 10 || hour > 18 || isWeekend) {
  alert("Το γραφείο είναι κλειστό.."); // Είναι το Σαββατοκύριακο
}
```

<h2 style="text-align: center;"> && </h2>

Ο τελεστής AND παριστάνεται με δύο τελείες `&&`:

```js
let result = a && b;
```

Στον κλασικό προγραμματισμό, το AND επιστρέφει true αν και οι δύο τελεστές είναι αληθείς και false αλλιώς:

```js
alert(true && true); // true
alert(false && true); // false
alert(true && false); // false
alert(false && false); // false
```

```js
let hour = 12;
let minute = 30;

if (hour == 12 && minute == 30) {
  alert("Είναι: 12:30");
}
```

---

<h2 style="text-align: center;"> ! </h2>

Ο τελεστής boolean ΟΧΙ αναπαρίσταται με ένα θαυμαστικό `!`.

Η σύνταξη είναι αρκετά απλή:

```js
let result = !value;
```

Ο τελεστής δέχεται ένα μόνο όρισμα και κάνει τα εξής:

1. Μετατρέπει το τελεστέο σε τύπο boolean: true/false.
2. Επιστρέφει την αντίστροφη τιμή.

```js
alert(!true); // false
alert(!0); // true
```
