<h2 style="text-align: center;"> Γενικά </h2>

Συχνά πρέπει να επαναλαμβάνουμε ενέργειες.

Για παράδειγμα, να βγάζουμε τα αγαθά από μια λίστα το ένα μετά το άλλο ή να εκτελούμε τον ίδιο κώδικα για κάθε αριθμό από το 1 έως το 10.

Οι βρόχοι είναι ένας τρόπος να επαναλαμβάνουμε τον ίδιο κώδικα πολλές φορές.

---

<h2 style="text-align: center;"> Η επανάληψη `while` </h2>

Ο επανάληψη `while` έχει την ακόλουθη σύνταξη:

```js
while (condition) {
  // code
}
```

Όσο η συνθήκη είναι αληθής, εκτελείται ο κώδικας από το σώμα του loop.

Για παράδειγμα, η παρακάτω επανάληψη εξάγει το `i` ενώ `i < 3`:

```js
let i = 0;
while (i < 3) {
  // 0, 1, 2
  alert(i);
  i++;
}
```

Μια μεμονωμένη εκτέλεση του σώματος του βρόχου ονομάζεται επανάληψη. Η επανάληψη στο παραπάνω παράδειγμα εκτελεί τρεις επαναλήψεις.

Αν το i++ έλειπε από το παραπάνω παράδειγμα, η επανάληψη του βρόχου θα επαναλαμβανόταν (θεωρητικά) για πάντα.

Οποιαδήποτε έκφραση ή μεταβλητή μπορεί να αποτελέσει συνθήκη επανάληψης, όχι μόνο συγκρίσεις: η συνθήκη αξιολογείται και μετατρέπεται σε boolean από την while.

Για παράδειγμα, ένας συντομότερος τρόπος για να γράψετε `while (i != 0)` είναι `while (i)`:

```js
let i = 3;
while (i) {
  // όταν το i γίνει 0, η συνθήκη γίνεται εσφαλμένη και η επανάληψη σταματάει
  alert(i);
  i--;
}
```

<h2 style="text-align: center;"> Η επανάληψη "for" </h2>

Ο βρόχος for είναι πιο πολύπλοκος, αλλά είναι επίσης ο πιο συχνά χρησιμοποιούμενος βρόχος.

Μοιάζει ως εξής:

```js
for (begin; condition; step) {
  // ... code ...
}
```

Ας μάθουμε τη σημασία αυτών των μερών με ένα παράδειγμα. Η παρακάτω επανάληψη εκτελεί την εντολή `alert(i)` για `i` από το 0``έως (αλλά **όχι** συμπεριλαμβανομένου) του`3`:

```js
for (let i = 0; i < 3; i++) {
  // shows 0, then 1, then 2
  alert(i);
}
```

Ο γενικός αλγόριθμος βρόχου λειτουργεί ως εξής:

```txt
Run begin
→ (if condition → run body and run step)
→ (if condition → run body and run step)
→ (if condition → run body and run step)
→ ...
```

Οποιοδήποτε μέρος του for μπορεί να παραλειφθεί.

Για παράδειγμα, μπορούμε να παραλείψουμε το begin αν δεν χρειάζεται να κάνουμε τίποτα στην αρχή της επανάληψης.

```js
let i = 0; // έχουμε ήδη δηλώσει και εκχωρήσει το i

for (; i < 3; i++) {
  // δεν χρειάζεται το "begin"
  alert(i); // 0, 1, 2
}
```

Μπορούμε επίσης να αφαιρέσουμε το τμήμα του βήματος:

```js
let i = 0;

for (; i < 3; ) {
  alert(i++);
}
```

Μπορούμε στην πραγματικότητα να αφαιρέσουμε τα πάντα, δημιουργώντας έναν άπειρο βρόχο:

```js
for (;;) {
  // Code
}
```
