<h2 style="text-align: center;"> Δηλώσεις </h2>

Οι δηλώσεις είναι συντακτικές δομές και εντολές που εκτελούν ενέργειες.  
Έχουμε ήδη δει μια δήλωση, την ```alert('Hello, world!')```, η οποία εμφανίζει το μήνυμα "Hello, world!".  
Μπορούμε να έχουμε όσες δηλώσεις θέλουμε στον κώδικά μας. Οι δηλώσεις μπορούν να διαχωρίζονται με ελληνικό ερωτηματικό, ```;```.  

Για παράδειγμα, εδώ χωρίζουμε το "Hello World" σε δύο ειδοποιήσεις:
```js
alert('Hello'); alert('World');
```

Συνήθως, οι εντολές γράφονται σε ξεχωριστές γραμμές για να κάνουν τον κώδικα πιο ευανάγνωστο:
```js
alert('Hello');
alert('World');
```  
  
Η άνω τελεία μπορεί να παραλειφθεί στις περισσότερες περιπτώσεις όταν υπάρχει αλλαγή γραμμής.  
Αυτό θα μπορούσε επίσης να λειτουργήσει:  
```js
alert('Hello')
alert('World')
```
  
Εδώ, η JavaScript ερμηνεύει τη διακοπή γραμμής ως "implicit" άνω τελεία.  
Στις περισσότερες περιπτώσεις, μια νέα γραμμή υπονοεί μια άνω τελεία.  
**Αλλά "στις περισσότερες περιπτώσεις" δεν σημαίνει "πάντα"!**  
Υπάρχουν περιπτώσεις όπου μια νέα γραμμή δεν σημαίνει άνω τελεία.

Για παράδειγμα:  
```js

1
+ 2);
```

<h2 style="text-align: center;">Σχόλια</h2>

Με την πάροδο του χρόνου, τα προγράμματα γίνονται όλο και πιο πολύπλοκα. Καθίσταται αναγκαία η προσθήκη σχολίων που περιγράφουν τι κάνει ο κώδικας και γιατί.  
Τα σχόλια μπορούν να τοποθετηθούν σε οποιοδήποτε σημείο ενός σεναρίου. Δεν επηρεάζουν την εκτέλεσή του, επειδή η μηχανή απλά τα αγνοεί.  
  
**Σχόλια μιας γραμμής ξεκινούν με δύο χαρακτήρες κάθετης προς τα εμπρός ```//```.**  
  
Το υπόλοιπο της γραμμής είναι ένα σχόλιο. Μπορεί να καταλαμβάνει μια ολόκληρη γραμμή από μόνο του ή να ακολουθεί μια δήλωση.  
  
Όπως εδώ:  
```js
// Αυτό το σχόλιο καταλαμβάνει μια δική του γραμμή
alert('Hello');

alert('World'); // Αυτό το σχόλιο ακολουθεί τη δήλωση
```  
  
**Τα σχόλια πολλαπλών γραμμών ξεκινούν με μια κάθετο προς τα εμπρός και έναν αστερίσκο ```/*``` και τελειώνουν με έναν αστερίσκο και μια κάθετο προς τα εμπρός ```*/```.**  
  
```js
/* Ένα παράδειγμα με δύο μηνύματα.
Αυτό είναι ένα σχόλιο πολλών γραμμών.
*/
alert('Hello');
alert('World');
```
Το περιεχόμενο των σχολίων αγνοείται, οπότε αν βάλουμε κώδικα μέσα σε ```/* ... */```, δεν θα εκτελεστεί.  
Μερικές φορές μπορεί να είναι χρήσιμο να απενεργοποιήσετε προσωρινά ένα μέρος του κώδικα:  
```js
/* Σχολιάζοντας τον κώδικα
alert('Hello');
*/
alert('World');
```

Παρακαλώ, μη διστάσετε να σχολιάσετε τον κωδικό σας.  
Τα σχόλια αυξάνουν το συνολικό αποτύπωμα του κώδικα, αλλά αυτό δεν είναι καθόλου πρόβλημα. Υπάρχουν πολλά εργαλεία που ελαχιστοποιούν τον κώδικα πριν από τη δημοσίευση σε έναν διακομιστή παραγωγής. Αφαιρούν τα σχόλια, ώστε να μην εμφανίζονται στα σενάρια εργασίας. Επομένως, τα σχόλια δεν έχουν καθόλου αρνητικές επιπτώσεις στην παραγωγή.
